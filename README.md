
# Arogya Sahaya Local - Digital Healthcare Companion

Arogya Sahaya Local is an Android application designed as a digital healthcare companion to assist elderly users, particularly in rural areas, with managing their health schedules. The application focuses on simple, high-accessibility UI components to track medication schedules and vital health metrics locally and reliably.

## 🚀 Features

* **Medication Scheduling & Reminders:** Helps users manage daily prescriptions with clear, easy-to-read interfaces.
* **Vital Tracking:** Allows users or caregivers to input and monitor core health metrics over time.
* **Offline-First Architecture:** Ensures all critical patient data is stored securely and accessible instantly without requiring a continuous internet connection.

## 🛠️ Tech Stack & Architecture

This project is built using modern Android development best practices:

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (for a modern, responsive, and highly accessible user interface)
* **Local Database:** Room DB (for secure, offline-first data persistence)
* **Design Pattern:** Repository Pattern with MVVM (Model-View-ViewModel) to ensure separation of concerns and clean, maintainable code.

## ⚙️ Setup & Installation Instructions

To run and evaluate this project locally, follow these steps:

1. **Clone the Repository:**
```bash
git clone https://github.com/Ashu1234512345/Arogya_Sahaya_Local-healthcare78.git

```


2. **Open in Android Studio:**
* Open Android Studio.
* Select **File -> Open** and navigate to the cloned project directory.


3. **Sync Gradle:**
* Allow Android Studio to complete the Gradle sync and download necessary dependencies automatically.


4. **Run the Application:**
* Select an Android Virtual Device (Emulator) or connect a physical device via USB debugging.
* Click the **Run** button (green play icon) in the top toolbar.



## 📌 Requirements

* **Minimum SDK:** API 26 (Android 8.0)
* **Target SDK:** API 34 (Android 14.0)
* **Build System:** Gradle (Kotlin DSL)
